Tidyzoning fromJSON()

Using the “rjson” package to turn the GeoJSON file into a nested list is really nice as well.

[[1]]
[[1]]$unit
[1] "feet"

[[1]]$use_name
[1] "1_family"

[[1]]$min_val
[[1]]$min_val$expression
[1] "50"
[[1]]
[[1]]$unit
[1] "square feet"

[[1]]$use_name
[1] "2_family" "3_family" "4_family"

[[1]]$min_val
[[1]]$min_val[[1]]
[[1]]$min_val[[1]]$conditions
[1] "bedrooms = 0"

[[1]]$min_val[[1]]$expression
[1] 500


[[1]]$min_val[[2]]
[[1]]$min_val[[2]]$conditions
[1] "bedrooms = 1"

[[1]]$min_val[[2]]$expression
[1] 700


[[1]]$min_val[[3]]
[[1]]$min_val[[3]]$conditions
[1] "bedrooms = 2"

[[1]]$min_val[[3]]$expression
[1] 900


[[1]]$min_val[[4]]
[[1]]$min_val[[4]]$conditions
[1] "bedrooms >= 3"

[[1]]$min_val[[4]]$expression
[1] 1000
