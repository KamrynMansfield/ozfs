find_district_idx()

The find_district_idx() function takes a tidyparcel and a tidyzoning dataframe and returns the index of the district in the tidyzoning dataframe that contains the tidyparcel.

Inputs

tidyparcel: This is data imported from a GeoJSON file with a feature for each edge of the parcel and a feature for the centroid of the parcel. The table below is an exmaple of the information that the GeoJSON contains for each parcel. The figure gives a visual of the example parcel.

The Function

find_district_idx <- function(tidyparcel, tidyzoning){
  # lists TRUE/FALSE to indicate which tidyzoning geometries cover the parcel centroid
  contains <- st_contains(st_make_valid(tidyzoning), tail(tidyparcel,1), sparse = F)
  # gives the index of the district that covers the centroid
  idx <- as.numeric(row.names(tidyzoning[contains, ]))
  
  # returns NA if there isn't a unique district covering the parcel centroid
  if (length(idx) == 1){
    print(idx)
  } else {
    print(NA)
  }
}
