find_bldg_type()

This is a very simple function that will count the number of rooms and give one of 5 outputs.

Possible.Outputs Description
1-family 1-family residence
2-family 2-family residence
3-family 3-family residence
4-family 4-family residence or greater
Other The number of units were not recorded 
find_bldg_type <- function(tidybuilding){
  possible_columns <- c("units_0bed",
                        "units_1bed",
                        "units_2bed",
                        "units_3bed",
                        "units_4bed")
  
  tidybuilding <- tidybuilding[,names(tidybuilding) %in% possible_columns] |>
    st_drop_geometry()

  row_sum <- rowSums(tidybuilding)
  
  if (row_sum %in% c(1,2,3)){
    paste0(row_sum[[1]], "_family")
  } else if (row_sum > 3){
    paste("4-family")
  } else {
    paste("other")
  }
}
