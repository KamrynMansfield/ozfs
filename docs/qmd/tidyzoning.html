The tidyzoning Package

Instalation

Load the devtools package

library(devtools)

Install package using install_github()

install_github("kamrynmansfield/tidyzoning")

Load the tidyzoning package

library(tidyzoning)

Information

To perform our analysis, we will create an R package called tidyzoning. The list below describes some of main things we hope the package will accomplish.

  • Take a building plan and find out where the zoning codes will allow it.
  • Take a parcel and find out what kind of building we can build on it.

Data Classes

In order to keep zoning, parcel, and building information uniform, the tidyzoning package has 4 data classes.

tidyzoning

An object representing zoning requirements and geometry for one or multiple zoning districts. It will be created using data stored in the Open-Zoning Feed Specification (OZFS) format.

More This data class has not been finalized yet, but I am thinking it could be a nested list mimicking the ozfs geojson structure along with a data frame that has the geometry of each district.

tidyparcel

An object representing geometry and sides of one or more parcels.

Functions

Bellow are the functions I have thought of and the progress I have made to creating them.

find_district_idx()

Takes a parcel and a tidyzoning object and returns the index of the tidyzoning object in which the parcel is found.

Creation Progress

find_bldg_type()

Takes a tidybuilding and returns what type it is. (1-familyl, 2-family, etc.)

Creation Progress

check_land_use()

Takes a tidybuilding and a district from a tidyzoning object and returns a boolean indicating whether or not the building’s type is permitted in the zoning district.

Creation Progress

check_fl_area()

Takes a tidybuilding and a district from a tidyzoning object and returns a boolean indicating whether or not the building’s floor area is permitted in the zoning district.

Creation Progress

check_footprint()

Takes a tidybuilding and a tidyparcel and returns a boolean indicating whether or not the building will fit in the parcel.

Creation Progress

check_parking()

Takes a tidybuilding, a tidyparcel, and a district from a tidyzoning object and returns a boolean indicating whether or not there is room for the required parking of the building.

Creation Progress
